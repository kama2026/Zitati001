# Random Quote Generator - Генератор случайных цитат

## Информация об авторе

**Фамилия и Имя:** Мусаев Камиль
**Дата создания:** 01.05.2026

## Описание

Приложение «Random Quote Generator» — это GUI‑приложение на Python, которое позволяет:

* генерировать случайные цитаты из предопределённого списка;
* просматривать историю сгенерированных цитат;
* фильтровать цитаты по автору и теме;
* добавлять новые цитаты (с проверкой корректности ввода);
* сохранять историю и цитаты в JSON‑файл между запусками приложения.

## Функциональность

- Генерация цитат: кнопка «Сгенерировать цитату» выбирает случайную цитату из списка (с учётом активных фильтров).
- История: отображает последние 20 сгенерированных цитат в прокручиваемом текстовом поле.
- Фильтрация: возможность фильтрации цитат по автору и теме через выпадающие списки.
- Сохранение данных: история и список цитат автоматически загружаются при запуске и сохраняются при нажатии кнопки «Сохранить данные».
- Валидация: проверка на пустые поля при добавлении новых цитат.


## Использование
1.  Генерация цитаты: нажмите кнопку «Сгенерировать цитату», чтобы увидеть случайную цитату, её автора и тему.

2.  Просмотр истории: история последних 20 цитат отображается в нижней части окна.

3.  Фильтрация:
- выберите автора из выпадающего списка «Фильтр по автору»;
- выберите тему из выпадающего списка «Фильтр по теме»;
- сгенерируйте новую цитату — она будет выбрана из отфильтрованного списка.

3.  Управление историей:
- «Очистить историю» — удаляет всю историю сгенерированных цитат из текущей сессии;
- «Сохранить данные» — сохраняет текущий список цитат и историю в файл quotes_data.json.
